# Phase II - Full-Stack Web Application Overview

## Project Name
Evolution-Todo Phase II: Multi-User Web Application

## Purpose
Transform the console todo app into a modern web application with user authentication and database persistence.

## Current Phase
**Phase II** - Full-Stack Web Application (Due: 6 PM Today)

## Tech Stack
- **Frontend**: Next.js 16 (App Router), Tailwind CSS
- **Backend**: FastAPI, SQLModel
- **Database**: SQLite (dev) / PostgreSQL (production)
- **Auth**: JWT tokens
- **Deployment**: Vercel (frontend), Railway (backend) - if time permits

## Core Features (Today's Scope)
1. ✅ User Registration (signup)
2. ✅ User Login (signin with JWT)
3. ✅ Add Task (authenticated)
4. ✅ View Tasks (user's own tasks only)
5. ✅ Update Task (title, description, status)
6. ✅ Delete Task
7. ✅ Logout
